Clean Architecture, Zero Decisions

Domain + Application

business/

Pure business logic. No dependencies on databases or frameworks. Contains your Entities, Value Objects, and Use Cases.

Adapters

infrastructure/

Database implementations (SQLx), external API clients, and messaging services. Where technical details live.

HTTP API

presentation/

Poem-openapi routes, CLI commands, and DTO mapping. The entry point for your system's users.

Quick Start

01

Install the global CLI tool via Cargo.

cargo install puerto
02

Bootstrap a new project with DDD structure.

puerto new my-app
03

Scaffold a new resource across all layers.

puerto generate scaffold User
